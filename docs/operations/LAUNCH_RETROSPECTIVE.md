# Launch Retrospective — Viewpulse

**Date:** 2026-04-05
**Stage:** Launch → Distribution

---

## What Worked

- Privacy-first positioning (data never leaves browser)
- BYOK AI model (no server-side AI costs)
- Games as engagement loop (Channel Duel, Rewatch Duel)
- Simple upload → instant insight flow
- GitHub repo as source of truth

## What Didn't

- No paid tier defined
- No referral partner pipeline
- No email capture for follow-up
- No landing page separate from app (Google Ads need conversion page)
- No social proof / testimonials

## Learnings

- Free tool generates organic traffic — distribution is the moat
- Watch history is more revealing than users expect — lead with the "scary accurate" hook
- Games keep users engaged longer — good for virality
- Referral model requires critical mass of professionals first

## Next Steps (Distribution)

1. Recruit first 5-10 approved mentors (therapists, career coaches)
2. Build referral matching flow into app
3. Set up Postgres CRM for partner tracking
4. Launch "Founding Mentor" program for first partners
5. Add email capture for user follow-up
